## Base URL

All API requests should be made to:

```bash
https://api.invaro.ai/api/v1
```

## Authentication

All API endpoints require authentication using Bearer tokens. You'll need to include your API key in the Authorization header:

```bash
Authorization: Bearer your_api_key_here
```

Get your API key from the [Invaro Dashboard](https://dashboard.invaro.ai).

## Quick Example

Here's a quick example of uploading and processing a bank statement:

```bash
# 1. Upload the file
curl -X POST "https://api.invaro.ai/api/v1/parse/upload" \
  -H "Authorization: Bearer your_api_key" \
  -F "files=@statement.pdf"

# 2. Process the statement
curl -X POST "https://api.invaro.ai/api/v1/parse/statements" \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "file_ids": ["1234567890_statement.pdf"]
  }'
```
